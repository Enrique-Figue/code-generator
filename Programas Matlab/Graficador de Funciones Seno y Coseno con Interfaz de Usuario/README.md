# 📊 Graficador de Funciones Seno y Coseno con Interfaz de Usuario 📈

¡Bienvenido al **Graficador de Funciones Seno y Coseno**! Este programa en MATLAB permite visualizar y personalizar gráficas de funciones trigonométricas mediante una interfaz gráfica intuitiva.

## 🚀 Funcionalidades
- **Interfaz gráfica interactiva** para ajustar parámetros.
- **Personalización** de amplitud, frecuencia, fase y color de las funciones.
- **Graficación simultánea** de seno y coseno en un mismo plano.
- **Actualización dinámica** al modificar valores.
- **Exportación** de gráficas como imágenes PNG.

## 📖 Instrucciones de Uso
1. **Ejecutar el programa**:
   - Abre MATLAB y navega al directorio del proyecto.
   - Ejecuta `Graficador_Seno_Coseno.m`.

2. **Configurar parámetros**:
   - **Amplitud**: Desliza el control o ingresa un valor numérico.
   - **Frecuencia**: Modifica la cantidad de ciclos de la función.
   - **Fase**: Ajusta el desplazamiento horizontal.
   - **Color**: Selecciona un color para cada función desde el menú desplegable.

3. **Graficar**:
   - Marca las casillas de **Seno** y/o **Coseno**.
   - Presiona el botón **"Graficar"** para visualizar las funciones.

4. **Exportar**:
   - Usa el botón **"Exportar gráfica"** para guardar la imagen en PNG.

## 🛠 Parámetros Ajustables
| Parámetro   | Descripción                          | Valor por Defecto |
|-------------|--------------------------------------|-------------------|
| Amplitud    | Altura máxima de la onda             | 1                 |
| Frecuencia  | Número de ciclos en el intervalo     | 1                 |
| Fase        | Desplazamiento horizontal (radianes) | 0                 |
| Color       | Color de la curva (16 opciones)      | Azul (Seno) / Rojo (Coseno) |

## 🔄 Ejemplo de Uso
1. Configura una función seno con:
   - Amplitud: 2
   - Frecuencia: 3
   - Fase: π/2
   - Color: Verde
2. Configura una función coseno con:
   - Amplitud: 1.5
   - Frecuencia: 2
   - Color: Magenta
3. Presiona **"Graficar"** para ver ambas funciones en acción.
