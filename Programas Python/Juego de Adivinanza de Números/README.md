# 🔢 Juego de Adivinanza de Números en Python 🎮

¡Bienvenido al **Juego de Adivinanza de Números**! Un programa interactivo donde pondrás a prueba tu intuición para adivinar un número secreto. 🌟

## 🚀 ¿Qué hace este programa?
- Genera un número aleatorio entre 1 y 100.
- Te da pistas (🔺 mayor o 🔻 menor) según tus intentos.
- ¡Gana puntos mientras más rápido adivines! 🏆
- Interfaz colorida y amigable en la terminal. 🌈

## 📥 Requisitos
- Python 3.x instalado.
- Módulo `random` (viene con Python por defecto).

## ⚙️ Instalación
1. Clona el repositorio o descarga el archivo `adivinanza.py`.
2. Abre una terminal en la carpeta del proyecto.
