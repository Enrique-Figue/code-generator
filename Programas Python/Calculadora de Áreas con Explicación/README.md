# 📐 Calculadora de Áreas con Explicación 🧮

¡Calcula áreas de figuras geométricas y entiende el proceso paso a paso! Este programa te permite seleccionar una figura, ingresar sus medidas y obtener el resultado junto con una explicación detallada del cálculo.

## 🚀 Características
- **Figuras disponibles**:
  - Círculo ⚪
  - Triángulo 🔺
  - Cuadrado ◼️
  - Rectángulo ▭
  - Pentágono regular ⬠
- **Explicación detallada**: Muestra las fórmulas utilizadas y los pasos matemáticos.
- **Interfaz intuitiva**: Menú interactivo para guiar al usuario.

## 📦 Requisitos
- Python 3.6 o superior.
- Biblioteca `math` (viene incluida en Python).
