# Calculadora Básica Interactiva en Python 🧮

¡Una calculadora intuitiva en terminal para operaciones matemáticas esenciales! Desarrollada en Python con ❤️.

## 🌟 Funcionalidades Principales

- **Operaciones disponibles:**
  - ✅ **Suma** (`+`)
  - ✅ **Resta** (`-`)
  - ✅ **Multiplicación** (`×`)
  - ✅ **División** (`÷`)
- 🔄 **Modo interactivo** con confirmación para continuar
- 📊 **Historial automático** de últimas 5 operaciones
- 🚨 **Protección contra errores:**
  - Entradas no numéricas
  - División por cero
  - Opciones de menú inválidas
